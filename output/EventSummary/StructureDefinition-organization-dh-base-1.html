SD.2 StructureDefinition-organization-dh-base-1

Base Organization [Draft]

Implementation guidance

For the supported usage scenarios in this implementation guide it is recommended that:

  • an Australian address conforms to the AU Base Address Profile

SD.2.1 Formal Views of Profile Content

The official URL for this profile is: 

http://ns.electronichealth.net.au/ci/fhir/4.0/StructureDefinition/organization-dh-base-1

The purpose of this profile is to define a representation of an organisation in an Australian context.

Must Support

In the context of this profile Must Support SHALL be interpreted as follows:

  • The system SHALL be able to include the element when authoring a resource where data is available

  • The system SHALL be able to store and retrieve the element

  • The system SHALL be able to process resources containing the element

This profile builds on AUBaseOrganisation.

This profile was published on Thu Apr 09 11:10:29 AEST 2020 as a draft by Australian Digital Health Agency.
